# <img src="assets/icon.png" width="32" height="32" valign="middle"> Trace

[![CI](https://github.com/studio2201/trace/actions/workflows/ci.yml/badge.svg)](https://github.com/studio2201/trace/actions/workflows/ci.yml)

Network diagnostic, WHOIS, IP, and ASN lookup web utility in Rust.

## Quick Start

### Self-Hosting (Docker)
Pull and run the official Docker container:
```bash
docker run -d -p 4404:4404 -v /path/to/appdata:/app/data ghcr.io/studio2201/trace:latest
```
